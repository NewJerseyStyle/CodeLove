# Examples 目錄說明

此目錄包含範例和參考代碼，不會被主劇情調用。

## 目的

- 提供實現範例供開發者參考
- 展示特定功能的完整實現
- 測試和實驗性代碼
- 存檔舊的實現方式

## 使用規則

1. **不要在主劇情中跳轉到這些標籤**
   - 主劇情應使用通用系統
   - 這些代碼僅作為參考

2. **標籤命名約定**
   - 必須使用 `example_` 前綴
   - 格式：`example_<功能名稱>` 或 `example_<章節>_<功能>`

3. **文件結構**
   ```
   rpy/examples/
   ├── README.md                    # 本說明文件
   ├── example_c01_algorithm_interaction.rpy  # C_01 章節範例
   └── example_*.rpy              # 其他範例
   ```

## Linter 處理

- `rpy/examples/` 目錄下的所有 `.rpy` 文件會被自動忽略
- 不會進行標籤分類和連接性驗證
- 這確保範例代碼不會產生錯誤或警告

## 添加新範例

1. 在此目錄創建新的 `.rpy` 文件
2. 使用 `example_` 前綴命名標籤
3. 添加清晰的註解說明用途
4. 如有相關文檔，在文檔目錄創建對應的 `.md` 文件
5. 更新本文件的「當前範例」表格

## 注意事項

- 範例代碼不保證與主劇情同步
- 範例代碼可能包含過時的做法
- 使用範例前請確認相關的通用系統是否已存在
- 如找到更好的實現方式，請更新範例
